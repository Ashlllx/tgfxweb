PAGViewer发布说明

最新版本（4.3.162）

一、修复问题：

1.1 导出插件：对AE2023之前的版本做兼容性适配

历史版本（4.3.158）

一、修复问题：

1.1 导出插件：修复了inter芯片的macOS中导出插件崩溃的问题。

历史版本（4.3.146）

一、增加功能：

1.1 PAGViewer：增加了“协议与声明”界面，用户首次使用时需要同意《软件许可及服务协议》与《隐私保护声明》条款。

二、优化：

2.1 PAGViewer：原生支持macOS ARM架构。
2.2 导出插件：更新AE SDK，适配AE新的轨道遮罩设置逻辑。

三、修复问题：

3.1 PAGViewer：修复了图层编辑时index可能有误的问题。
3.2 PAGViewer：修复了一些文本显示异常的问题。
3.3 PAGViewer：修复了可编辑图层的缩放模式不生效的问题。
3.4 导出插件：修复了windows下导出BMP文件失败的问题。
3.5 导出插件：修复了macOS下向临时文件夹写入文件失败的问题。
3.6 导出插件：修复了AE设计文件父子图层维度不一致，导致导出为pag文件后子图层位置异常的问题。
3.7 导出插件：修复了AE 3D图层导出为pag文件后可见范围异常的问题。
3.8 导出插件：修复了偶现的数组越界问题。

历史版本（4.3.6）

一、增加功能：

1.1 增加支持了AE图层样式：描边
1.2 增加支持的AE图层样式：外发光
1.3 PAGViewer：主界面上增加了显示当前帧号/总帧数
1.4 libpag内核更新为4.3.3版本

二、优化：

2.1 导出插件：将导出PAG时BMP视频编码更改为采用调用外部独立进程的方式。
2.2 PAGViewer：将添加水印里BMP视频编码更改为采用调用外部独立进程的方式。
2.3 导出插件：优化了导出面板设置页里的文本设置与占位图设置。
2.4 导出插件：优化了插件选项配置面板。
2.5 PAGViewer：图层编辑里不再显示不可编辑文本/图片。
2.6 PAGViewer：TagLevel 更换为显示素材所需要的最低版本号。
2.7 PAGViewer：windows 英文菜单显示。

三、修复问题：

3.1 PAGViewer：修复了windows上偶现卡住的问题。
3.2 PAGViewer：修复修改完 PAG 图层名称之后, 无法再次打开该 PAG 的问题

历史版本（4.2.63）

一、修复问题：

1.1 PAGViewer：修复首次打开 PAG 文件第一帧蓝屏闪烁的问题。
1.2 PAGViewer：修复替换照片后无法撤销的问题。
1.3 PAGViewer：修复 Effect 图层含有缩放时效果不对的问题。

历史版本（4.2.56）

一、增加功能：

1.1 PAGViewer：补齐了windows上自动检测并提示更新AE插件的功能。
1.2 图层变换支持3D属性。
1.3 支持摄像机图层。
1.4 增加两个效果“色相饱和度”、“亮度对比度”。
1.5 PAGViewer：PAG企业版功能发布。
1.6 PAGViewer：证书管理。
1.7 PAGViewer：素材加水印。

二、修复问题：

2.1 修复3D图层“方向”属性不准确的问题。
2.2 导出插件：修复导出偶现“解码失败”的问题。
2.3 导出插件：修复部分机型上打开"PAG File (Panel)..."时crash的问题。
2.4 导出插件：修复mac“勾选BMP合成，弹窗确认时选择取消，之后无法继续勾选”的问题。
2.5 PAGViewer：修复文件结构里不显示Point3D属性的问题。
2.6 PAGViewer：修复PAGViewer自带QT环境与本机安装QT环境冲突造成PAGViewer无法打开的问题。
2.7 PAGViewer：修复稳定性问题，解决部分机型启动crash的问题。
2.8 PAGViewer：修复带有时间变换图层替换视频后，视频播放不正确的问题。
2.9 PAGViewer：修复部分PAG文件替换视频后，音频播放不正确的问题。
2.10 PAGViewer：修复编译和安装阶段没有忽略RPATH导致的crash问题。
2.11 PAGViewer：修复macOS每次新打开PAG文件都新开窗口不用应用已有空窗口的问题。
2.12 PAGViewer：修复了windows上已是最新版仍然提示有更新版的问题。

历史版本（4.1.2）

一、增加功能：

1.1 导出插件：增加了对 mac M1 机型的支持（rosetta和非rosetta方式皆可）。

二、修复问题：

1.1 PAGViewer：修复图片作为TrackMatte使用时，替换图片会导致crash的问题。
1.2 PAGViewer：修复了替换图片时音频停止播放的问题。

历史版本（4.1.1）

一、新增功能：

1.1 导出插件和PAGViewer都支持切换中英文设置。
1.2 导出插件的导出面板可设置文本/图片是否可编辑。
1.3 语言增加了对英语的支持。
1.4 实现了关键帧属性“漂浮穿梭时间”。
1.5 PAG导出增加效果：色相/饱和度、亮度/对比度。

二、优化：

2.1 导出插件摒弃了一些与具体业务相关的检测（如videoTrack覆盖）。
2.2 完善了中英文文案。
2.3 优化PAGViewer版本更新的流程。
2.4 PAGViewer mac平台使用硬件解码。
2.5 导出字体文件避免重复拷贝。

三、修复问题：

3.1 替换图在多个时间点出现时，替换图片操作只能替换其中一个时间点的问题。
3.2 部分机器上evalScript执行jsx中失败的问题。
3.3 修复了文本图层带“渐变叠加”图层样式时导出卡住的问题。
3.4 修复了Mac 10.14等低版本上无法导出的问题。

历史版本（4.0.5）

一、修复问题：

1.1 Windows设置隐藏已知类型文件后缀时，某些机器上导出的文件名没有.pag后缀。
1.2 文本对齐方式为居中对齐和右对齐时，含有换行的文本位置发生了偏移。
1.3 部分还有文本图层的工程导出时卡住。
1.4 文字字体不存在时导出字体引起奔溃。
1.5 PAGViewer部分文字显示出现凸起边缘。
1.6 PAGViewer 在部分机型无法替换图片和缩略图。
1.7 导出面板搜索合成时合成名无法更新。

二、优化：

2.1 文本图层字体太小而缩放比例过大时，提醒警告。
2.2 PAGViewer mac平台使用硬件解码。
2.3 导出字体文件避免重复拷贝。

三、架构升级：

3.1 底层升级为libpag 4.0架构。
