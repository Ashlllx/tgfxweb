#### 最新版本（4.0.7）

#### 历史版本（4.0.5）
##### 一、修复问题：
1.1 Windows设置隐藏已知类型文件后缀时，某些机器上导出的文件名没有.pag后缀。<br/>
1.2 文本对齐方式为居中对齐和右对齐时，含有换行的文本位置发生了偏移。<br/>
1.3 部分还有文本图层的工程导出时卡住。<br/>
1.4 文字字体不存在时导出字体引起奔溃。<br/>
1.5 PAGViewer部分文字显示出现凸起边缘。<br/>
1.6 PAGViewer 在部分机型无法替换图片和缩略图。<br/>
1.7 导出面板搜索合成时合成名无法更新。<br/>
##### 二、优化：
2.1 文本图层字体太小而缩放比例过大时，提醒警告。<br/>
2.2 PAGViewer mac平台使用硬件解码。<br/>
2.3 导出字体文件避免重复拷贝。<br/>
##### 三、架构升级：
3.1 底层升级为libpag 4.0架构。<br/>
