## 背景
为了提升 pag 素材在终端的渲染性能，根据之前的一些经验，在 AE 导出插件中内置了以下检测规则，协助设计师设计出渲染性能较好的 pag 素材。
</br>
| 校验项 | 适用场景 |
| ------ | --------- |
| 自动检测到相邻的多个 BMP 预合成时，弹窗建议合并为一个 | 通用 |
| 自动检测到 ImageFillRule 加在非 Image 图层上时弹窗提示 | 通用 |
| 自动检测位图数目超过 50 个时弹窗提示 | 通用 |
| 自动检测 BMP 预合成数目超过 3 个时弹窗提示 | 通用 |
| 自动检测图层总数超过 60 时弹窗提示 | 通用 |
| 自动检测同一时刻效果和 layerStyle 峰值数目，超过 3 个时弹窗提示 | 通用 |
| 自动检测到 ImageFillRule 加在非 Image 图层上时弹窗提示 | 通用 |
| 自动检测到快放超过 8 倍速时弹窗提示 | 视频编辑 |
| 自动检测同一时刻 BMP 预合成峰值数目（重叠图层数），超过2个时弹窗提示 | 视频编辑 |
| 自动检测到不同 BMP 预合成图层引用同一个素材时弹窗提示。（单个素材仅被引用一次）| 视频编辑 |
| 自动检测在时间轴上出现某个 BMP 预合成图层的时间完全包含另一个 BMP 预合成时，<br/> BMP 预合成图层要连续，中间不能插入其它 BMP 预合成 | 视频编辑 |
| 自动检测到图片图层中需要填充的资源小于 0.5s 时弹窗提示 | 视频编辑 |
| 自动检测到 TimeRemap 为倒放（ endTime 小于 startTime）时弹窗提示 | 视频编辑 |
| 自动检测 UI 场景下含有 BMP 预合成时弹窗提示 | UI 动画 |
