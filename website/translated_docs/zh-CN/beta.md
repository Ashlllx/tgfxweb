### PAGViewer 版本

PAGViewer 有两个版本：Stable 和 Beta 版本。

Stable 版本对应 libpag 源码中的最新 release 分支，功能相对稳定。

Beta 版本对应 libpag 的 main 主开发分支，会包含最新的功能。如果你想体验最新的功能，欢迎使用 Beta 版本。

### 切换到 Beta 版本
1.  点击顶部菜单 PAGViewer-> 偏好设置
<img 
  src='/img/docs/beta_step1.jpg' 
  style='width: 700px; margin: 32px 0 48px 0' 
/>

2. 在弹窗中选择使用 Beta 版本
<img 
  src='/img/docs/beta_step2.jpg' 
  style='width: 700px; margin: 32px 0 48px 0' 
/>

3. 点击顶部菜单 PAGViewer-> 检查更新
<img 
  src='/img/docs/beta_step3.jpg' 
  style='width: 700px; margin: 32px 0 48px 0' 
/>

4. 更新后，点击顶部菜单 PAGViewer->关于 PAGViewer
<img 
  src='/img/docs/beta_step4.png' 
  style='width: 700px; margin: 32px 0 48px 0' 
/>
5. 版本变为 Beta 版本啦
<img 
  src='/img/docs/beta_step5.png' 
  style='width: 700px; margin: 32px 0 48px 0' 
/>
