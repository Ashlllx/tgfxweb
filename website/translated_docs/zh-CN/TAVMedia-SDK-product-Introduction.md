**TAVMedia SDK** 可提供多平台的视频编剪及模板制作能力，集成了剪辑、拼接、特效、转场、播放、导出等功能的客户端 SDK，帮助用户聚焦业务本身，快速轻松实现多平台（Andorid，iOS）的视频编剪和模板应用。

### 短视频制作工具

可利用模板 SDK 开发短视频制作工具，让用户快速制作视频。</br>

### 模块化包装能力

为广告行业提供模块化包装能力，帮助广告行业利用产品卖点和特效预制作广告大片。</br>
<img src="/img/docs/changjing22.jpg" alt="img" style="margin: 32px 0 48px 0;" /></br>

### 视频制作能力
