####  PAG 文件

一种自研的二进制动效文件格式，设计师在 AE 中设计好动效，可以通过 PAGExporter 导出

#### PAGExporter

一种 AE 导出插件，可以导出 PAG   文件

#### PAGViewer

一种桌面预览工具，支持预览 pag 文件，支持 macOS 和 Windows 两个平台

#### 矢量导出

导出插件 PAGExporter 默认的导出方式，该导出方式导出的文件极小，渲染性能较好，可以覆盖常用 AE 特性，但无法涵盖所有的 AE 特性，如粒子效果、第三方插件特效等

#### BMP 预合成导出

导出插件 PAGExporter 中的一种导出方式，支持所有的 AE 特性导出，但不支持运行时编辑，原理是针对渲染的画面截图编码成视频序列或图片序列，适用于使用了不能直接导出的 AE 特性的场景


#### 混合导出

导出插件 PAGExporter  中矢量导出 和 BMP 预合成相结合，BMP 预合成不在根节点中选择即可，既可以保持动效的可编辑性，又可以支持全 AE 特性

#### BMP 预合成

有两种编码格式：视频序列帧和图片序列帧，默认为视频序列帧，可以在插件选项配置面板中设置导出编码格式，在 PAGExporter 对特定合成或图层勾选，该合成或图层在 pag 文件中将会以 BMP 预合成的格式存在。


#### 位图序列帧

BMP 预合成的一种编码格式，在插件选项配置面板中设置，设置后 BMP 勾选的预合成将会以位图序列的格式导出

#### 视频序列帧

BMP 预合成的一种编码格式，为默认值，在插件选项配置面板中设置，设置后 BMP 勾选的预合成将会以视频序列的格式导出

#### 占位图

指的是 pag 文件中图片图层的默认图片，该图片支持通过 API 接口替换
