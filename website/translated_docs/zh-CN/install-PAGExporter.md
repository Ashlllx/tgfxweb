title: 安装 AE 导出插件
---
---

设计师在 AE 中完成动效设计后，需要通过导出插件 PAGExporter 导出[<font color=blue> PAG 文件</font>](/docs/terms.html#pag-文件)

### 准备工作：
> 1，确保已经安装了 AE（Adobe After Effects）（AE 2018 及以上版本皆可）<br/>
> 2，确保已经安装 PAGViewer。[<font color=blue>如何安装 PAGViewer</font>](/docs/install.html)<br/>

### 安装 AE 导出插件
1，如果 AE 正在运行，请关闭后再安装<br/>
2，打开 PAGViewer。PAGViewer 将自动检测是否需要安装/更新 AE 导出插件，按提示安装即可。
<img 
  alt='自动安装AE插件截图' 
  src='/img/docs/install-PAGExporter1.jpg' 
  style='width: 500px; margin: 24px 0 48px 0' 
/>

---

3，如果 PAGViewer 没有自动提示安装，则可以点击触发安装： <br/>

+ macOS：点击菜单栏 "PAGViewer" -> "安装AE插件" <br/>
+ Windows：点击菜单栏 "文件"" -> "安装AE插件" <br/>

<img 
  alt='点击触发安装AE插件截图' 
  src='/img/docs/install-PAGExporter2.jpg' 
  style='width: 500px; margin: 24px 0 48px 0' 
/>

---
### 验证
  打开 AE 软件，如果在菜单项中将会看到：“文件” -> “导出” -> “PAG File...”，则说明已经成功安装。

### 手动安装
  如果自动安装失败，Windows 平台可以选择[<font color=blue>手动安装</font>](/docs/manual-install-PAGExporter.html)<br/>
