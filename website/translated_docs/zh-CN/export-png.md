桌面端预览工具 PAGViewer 支持导出正在预览的 PAG 文件的当前帧、整个文件的图片序列，可用于查看分析问题、基于图片序列转 gif 等。

## 导出方法
1. 正常打开某 PAG 文件
2. 如果需要导出某一帧，则请先（通过空格和左右方向键控制）暂停在该帧
3. 点击顶部菜单 "File -> 导出 -> 保存当前帧为 PNG"

<img
src='/img/docs/export_png.png'
style='width: 640px; margin: 32px 0 48px 0'
/>

4. 如果需要导出图片序列，选择"导出 PNG 序列帧" 或 "导出 APNG"
