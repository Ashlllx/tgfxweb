## 使用场景
添加水印功能可以有效防止动效被盗用或者未授权转载，适用于社交媒体分享。</br>

## 水印效果展示
添加水印后，PAG 文件画面中会出现用户自定义的水印图案或文字。</br></br>

## 如何添加水印
### 1. 功能入口
点击菜单栏——添加水印，即可打开添加水印弹窗。</br>
<img src="/img/docs/watermark3.png" alt="img" style="zoom:50%;margin: 32px 0 48px 0;" /></br></br>
### 2. 设置水印类型
支持两种水印类型：图片水印和文字水印。</br>
<img src="/img/docs/watermark4.png" alt="img" style="zoom:40%;margin: 32px 0 48px 0;" /></br></br>
#### 图片水印
用户可以在本地文件中选择想要作为水印的图片，支持 JPG、JPEG、PNG 等图片格式。</br></br>
#### 文字水印
用户可以输入 10 个汉字或 20 个英文字符以内的文字水印。</br></br>
### 3. 添加 PAG 文件
打开【添加水印】弹窗后，文件列表默认加入当前显示的 PAG 文件，</br>
若想批量添加水印，可直接点击【添加按钮】添加 PAG 文件或直接将文件拖入到文件列表中。</br>
<img src="/img/docs/watermark5.png" alt="img" style="zoom:40%;margin: 32px 0 48px 0;" /></br></br>
### 4. 导出
用户完成水印添加、选择合适的导出路径后，点击右上角【导出】按钮即可获得带水印的 PAG 文件。</br>
