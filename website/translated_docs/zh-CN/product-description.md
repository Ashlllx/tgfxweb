## PAG 应用场景

PAG 动效工作流解决方案广泛应用于各种动效场景，如 UI 动画、贴纸动效、视频模版、拍摄挂件、直播特效、视频广告、游戏战报等，覆盖 Android、iOS、macOS、Windows、Linux、Web 和微信小程序等常用平台。

## PAG 核心价值
PAG 的目标是降低或消除动效相关的研发成本，打通设计师创作到素材上线的自动化流程。
<img 
  src='/img/docs/pag-introduction/4.png' 
  style='width: 100%; margin: 32px 0 48px 0;max-width: 100%' 
/>
<img 
  src='/img/docs/pag-introduction/5.gif' 
  style='width: 100%; margin: 32px 0 48px 0;max-width: 100%' 
/>

设计师在 AE 中完成动效设计后，通过导出插件 PAGExporter 导出 PAG 文件，在桌面端通过 PAGViewer 预览效果和查看性能，确认效果无误后分发给各平台端通过 PAG SDK 完成动效的渲染展示。
